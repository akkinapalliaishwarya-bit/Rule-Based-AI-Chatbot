# 🤖 Rule-Based AI Chatbot

## 📌 Overview

This project is a **Rule-Based AI Chatbot** developed as part of an Artificial Intelligence internship program. The chatbot simulates basic human conversation by responding to predefined user inputs using conditional logic (`if-elif-else`).

This project focuses on building a strong foundation in **control flow, logical reasoning, and basic AI concepts** without using machine learning techniques.

---

## 🎯 Objectives

* To understand the fundamentals of Artificial Intelligence systems
* To implement decision-making using conditional statements
* To design a simple conversational interface
* To simulate human-like interaction using rule-based logic

---

## 🚀 Features

* 👋 Handles greeting messages (Hi, Hello, Hey)
* 💬 Responds to predefined user queries
* 🔁 Runs in a continuous loop for interaction
* ❌ Exit functionality (Bye, Exit, Quit)
* ⚠️ Handles unknown inputs gracefully

---

* **Concepts Used:**

## ⚙️ How It Works

The chatbot follows a simple rule-based approach:

1. Takes user input from the console
2. Converts input to lowercase for consistency
3. Matches input with predefined conditions
4. Returns a corresponding response
5. Repeats the process until the user exits
